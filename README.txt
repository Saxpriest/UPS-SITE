ParcelTrack Replica - UPS-style mockup

Files in zip:
 - public/index.html
 - public/styles.css
 - public/script.js
 - public/images/* (placeholders)
 - server.js
 - package.json

To run locally:
1. unzip parceltrack_replica.zip
2. cd parceltrack_replica
3. npm install
4. npm start
5. open http://localhost:3000

Replace images in public/images with licensed assets for exact visual match.
